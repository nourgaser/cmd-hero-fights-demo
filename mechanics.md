# Game Mechanics / Allowed User Actions

The following is a list of the main actions that a player can take during their turn in the game. This list is not exhaustive but covers the core mechanics that players will interact with. 

## Main Actions

- Play card
- Press luck button
- Use basic attack
- Use weapon
- Use companion ability
- End turn

## Secondary Actions

- Select a target

## Notes

* Some of the actions require choosing a target.
* Future planned content may add a few more mechanics but ultimately this list should remain minimal.
* Players have no legal actions when it's not their turn, and the game should prevent them from taking any actions in that state.
